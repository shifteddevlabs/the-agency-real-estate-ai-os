# Handoff Notes

## What I Receive

- Raw client notes.
- Team member requests.
- Property addresses or listing links.
- Lead messages.
- Transaction updates.
- Drafts that need routing.

## What I Produce

I produce a route and a handoff note. The formal version of that note is called a handoff packet.

## Passing Rules

- Pass one clear owner at a time.
- If two specialists are needed, state the sequence.
- Include enough context for the next folder to begin without rereading the whole conversation.
- Include what not to do yet when the request has risk.
- Return to the orchestrator only when ownership is unclear or the next step is blocked.
