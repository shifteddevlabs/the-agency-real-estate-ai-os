# Private Lead/Deal Logs

Save one short lead/deal log here for any active lead, client, property, or deal you want the AI to remember in a future chat.

If your AI app can write files, ask it to save the log here. If it cannot, ask it for the exact filename and text, then paste that into your CRM or deal notes.

Recommended filename:

```text
YYYY-MM-DD-client-or-address.md
```

Examples:

```text
2026-05-18-alex-east-austin-buyer.md
2026-05-18-4812-sample-bend.md
```

Real logs are private and ignored by Git.

Do not commit real client names, phone numbers, emails, deal terms, wire details, or private transaction notes.

If your team already uses a CRM, Google Drive deal folder, or transaction platform, that system is still the official record. This folder is only a memory aid you can paste into a new AI chat.
