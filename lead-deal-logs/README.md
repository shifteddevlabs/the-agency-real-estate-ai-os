# Private Lead/Deal Logs

Save one markdown lead/deal log per active lead, client, property, or deal in this folder when you are using The Agency locally.

Recommended filename:

```text
YYYY-MM-DD-client-or-address.md
```

Examples:

```text
2026-05-18-alex-east-austin-buyer.md
2026-05-18-4812-sample-bend.md
```

Real lead/deal logs are ignored by Git. Do not commit real client names, phone numbers, email addresses, deal terms, wire details, or private transaction notes.

If your team already uses a CRM, Google Drive deal folder, or transaction platform, that system remains the official record. This folder is only an AI memory snapshot you can paste into a new chat.
